# Caption and Table Checklist

- Every figure caption states setup, metric, and take-away.
- Every table has units or clear metric semantics.
- Higher/lower-is-better is obvious.
- Baselines are named consistently with text.
- Boldface or highlighting has a clear rule.
- Captions can be understood without hunting the main text.
- Dense tables are split if readers cannot find the intended comparison fast.
